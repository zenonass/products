**Install prerequisites:**

Using data/db.json file as json server for resources.  

npm i json-server  
npx json-server --watch data/db.json --port 8000  
npm start  


**Endpoints used:**  
GET all products "/"  
GET single product "/products/{id}"  
POST create new product "/create"
